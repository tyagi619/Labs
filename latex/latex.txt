\documentclass[OPTIONS]{CLASS}

		OPTIONS : 
		1-letterpaper,a4paper,legalpaper,a5paper,executivepaper,b5paper
		2-10pt,11pt,12pt
		3-onecolumn/twocolumn :1 column / 2 column pages
		4-oneside/twoside : Print on one side / both sides of paper
		5-openright/openany : IDetermines if a chapter should start on a right-hand page; default is openright for book, and openany for report
		6-landscape : Selects landscape format; default is portrait
		7- draft/final : Mark (draft) or do not mark (final) overfull boxes with a black box in the margin; default is final
		8-fleqn : Put displayed formulas flush left; default is centered
		9- leqno : Put equation numbers on the left side of equations; default is the right side
		10-openbib : Use “open” bibliography format
		11- titlepage/notitlepage : Specifies whether the title page is separate; default depends on the class

		**OPTION 3,4,5 are not used by slides
		**Slides class offers the option 'clock' for printing the time at the bottom of each note
